# Personal Portfolio Website

## About
A clean, responsive personal portfolio built from scratch using pure HTML, CSS, and Vanilla JavaScript. It showcases my background as a Computer Science undergraduate, the technologies I’ve worked with, my academic journey, and the projects I’ve built.

---

## Features
- **Responsive Design** — Optimized for both mobile and desktop with a hamburger navigation menu and scroll lock functionality
- **Scroll Reveal Animations** — Sections animate into view using `IntersectionObserver`
- **Glass-Card UI** — Modern glassmorphism-inspired design with a custom education timeline
- **Hero Section** — Includes CTA buttons and an “Open to Opportunities” availability badge
- **Projects Showcase** — Project cards with technology tags for each build
- **Footer with Social Links** — Quick access to LinkedIn and GitHub profiles
